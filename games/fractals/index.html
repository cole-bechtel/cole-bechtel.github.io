Fractal Maker

By Cole Bechtel

Explanation:

Points is the number of anchor points that will "attract" the points that are drawn.
Speed is the number of points that are calculated and drawn each time the program loops.
Ratio is the percentage along the line between the previous point and the chosen anchor point that the new point will be drawn. The default ratio is 0.5, which will draw the new point at the midpoint between the previous point and the anchor point

Things to try:

Sierpinski triangle: 3 points with a ratio of 0.5
Shapes: n points with a ratio of 0.789 will result in a shape with n sides that is itself made of the same shape, and so on.
Pentagon: 5 points with a ratio of 0.5 will result in a pentagon that is riddled with holes.
